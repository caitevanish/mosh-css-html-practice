  • translateZ(): This hover property will move the element closer (+) or far (-) away from you. Must call the perspective function first!
  • RotateY() : Rotate the object around its center. To change that position, must use transform-origin: rule
  • Wrap elements in a container class and they will animate together

Make sure all elements are sharing the same perspective


.container {
perspective: 200px

5- Animations

Lorem ipsum, dolor sit amet consectetur adipisicing elit.

Animation for when you want to go beyond the basic transformations

    @keyframes:
  • animation has several frames
  • Call your animation anything (ie.pop)
  • Specify percentage of time of animation sequence

    • 0% = first frame
    25% = make slightly larger
    50% = rotate 45deg clockwise, also change background to tomato
    100% = rotate element to 0deg (back to normal position)
    .box element
  • animation-name:pop
  • animation-duration: 4s;
  • Animation starts on page refresh
  • animation-iteration-count: infinite
  • animation-timing-function: ease-out
  • animation-direction: alternate
