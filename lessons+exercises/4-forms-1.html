2-Creating a Basic Form

  • Don't use paragraph element, but label
  • Give input field a unique identifier(id), use the same value for the for attribute of label
  • Must supply a valid email address in the email fill
  • Vertical stacking, wrap the label and input in a div
  • Button types: Submit + Clear

3-Styling Forms

  • CSS: font-family, line-height (Usually in body, but in this case it's .lesson-3 class)
  • Make Label display:block; violating law of proximity. Must add vertical space between groups
  • Ordering of rules: have more generic rules on top, more specialized rules on the bottom
  • Add spacing around grouped items with padding; change the color of the border in input;
  • Don't want to blindly apply input rule to all inputs, only text fields; use selector! [type="text"].
    Add a second selector for the email.
    input[type='text'], input[type='email']
  • Use state to change the color of the border when it's selected/focused. Change the Outline to 0. Add box shadow!
  • Input: Add transition property!
    transition: border-color 0.15s, box-shadow 0.15s;
  • Style Buttons!
    button.lesson-3 {
    background: rgba(24, 117, 255, 0.75);
    color: #fff;
    border: 0;
    padding: 0.5rem 0.7rem;
    border-radius: 5px;
    outline: 0;
    }
