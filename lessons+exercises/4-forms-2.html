6- Data Lists

  • Provide list of suggestions: datalist and options
  • When text box is focused you see the list
  • Create own custom attribute iwth data-value="1"
  • turn off autocomplete value as input attribute
  • Datalist: Browser default styles will take over; set list attribute to an empty string. Use Javascript to manually display datalist

7- Drop-down Lists

  • Select: need name attribute when you are submitting form to server
  • To have a pre-selected item, use selectedattribute (Boolean value) < option value="1" selected>HTML/>
  • To select multiple items: multiple attribute in select tag
  • If too many items in list, categorize them with Option Group Element

8- Check Boxes

  • Tip! Type "input:" and you will see a list of different types of inputs!
  • Use utility class to put the label next to the checkbox
  • Stack elements vertically by wrapping in a div
  • Have check box checked by default (boolean attribute) in input tag: < input type="checkbox" id="back-end" checked>

9- Radio Buttons

  • Use name attribute in input tag to group multiple radio buttons
  • Pre-selected radio button: checked attribute in input
