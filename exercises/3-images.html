    Instructions for Data URLs:
  1. Go to a Image to Data Generator
  2. Copy Data URl code
  3. Paste into markup

Data URL comes straight from memoty. Our browser did not send a separate HTTP request to download the image because it's hodedinto the html. We can achieve faster load time. The size of embedded code is always larger than the original file. Likely will not use it, but good to know what it is and how to use it when some websites are employing this method

Supporting high density screens

    Terms:
  • Physical resolution: (ie. 960 x 640)the acutal number of pixels on device
  • Logical resolution: (480 x 320) CSS always based on logical res.
  • Device Pixel Ratio (DPR): 2 px
  • High Density: DPR >1
  • 1x, 2x, 3x should cover most phone resolutions

Resizing in Photoshop:

ALWAYS START WITH LARGEST IMAGE
    3x image size:
  1. Open Image
  2. Image menu > Image size
  3. Change width to 1200px (400px x 3)
  4. Save file "name@3x.jpg"
  5. Compress image to 7 quality
    2x image size:
  1. Image menu > Image size
  2. Change width to 800px (400px x 2)
  3. Save file "name@2x.jpg"
  4. Compress image to 7 quality
  5. etc.

For high-def images, must provide source attribute

< img src="/images/meal.jpg"
alt="Bowl of Salmon and curry"
class="meal4"
srcset="
/images/meal.jpg 1x,
/images/meal@2x.jpg 2x,
/images/meal@3x.jpg 3x
"
/>

This technique is for fixed sized images. Does not need to be used for all images, just the really good looking ones.

Resolution Switching

Different resloutions for different sized devices

< img src="/images/meal.jpg"
alt="Bowl of Salmon and curry"
class="meal4"
srcset="
/images/meal.jpg 400vw,
/images/meal@2x.jpg 800vw,
/images/meal@3x.jpg 1200vw
"
sizes="
(max-width: 500px) 100 vw,
(max-width: 700px) 50vw,
33vw
" />

*Must be vw!
Don't use media queries for different sizes
The breakpoints in the sizes attribute here are arbitrary!
Always start with mobile first, since that is what the user will likely see first

Check to see what breakpoint sizes should be at Responsive Image Breakpoints Generator
